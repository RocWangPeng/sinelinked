# 晟联保信平台

基于保险，法律，广告的整合传播平台
---
## 项目架构
- PC端
    - **开发模式**：前后端分离
    - **前后通信**:axios
    - **PC端UI框架**：bootstarp、layui
- 移动端
    -  **开发模式**：前后端分离
    -  **前端框架**：VUE
    -  **css预处理**：less
    - **构建工具**：webpack
